# Results Folder

This folder contains a small set of final exported CSV files copied from the original experiment artifacts.

## Included Files

- `clean_vs_lowlight_baseline_5k.csv`
  - baseline clean-detector evaluation on one balanced 5k subset
- `final_comparison_seed42.csv`
  - final three-model comparison on the first balanced 5k test subset
- `final_comparison_seed314.csv`
  - final three-model comparison on the second balanced 5k test subset

## Why These Files Are Included

They provide compact evidence for the final report without requiring the full training workspace or all intermediate evaluation folders.

The README summary numbers are based on these exported comparison tables.
